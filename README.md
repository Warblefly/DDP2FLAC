# DDP2FLAC
### by John Warburton, Institute of Sound Recording, University of Surrey

This is the start of a program that opens a DDP folder, and creates separate FLAC files from each track, ready for importing into workstations etc.

Currently, the program parses for album titles as CD text, naming the files appropriately. Soon, a best-guess will be taken based on folder names if no CD-TEXT is available.

The next revision will allow full spanning of all folders contained within other folders or even a .zip archive, so that many zipped together DDPs can be converted in a single operation.
